## Installation

1. Clone the repository:
```bash
git clone [repository-url]
cd ai-pentest-app
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Configure environment variables:
```bash
cp .env.example .env.local
```

4. Start the development server:
```bash
npm run dev
# or
yarn dev
```
